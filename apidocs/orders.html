API Documentation

GET /orders List all orders

Parameters:

  • limit (query, int) - Limit the number of results
  • offset (query, int) - Offset the results

Responses:

  • 200 - array: OK
  • 400 - object: Invalid request
POST /orders Create a new order

Responses:

  • 201 - object: Order successfully created
  • 403 - object: Unauthorized
GET /orders/{id} Get an order by ID

Parameters:

  • id (path, int) - The ID of the order

Responses:

  • 200 - object: OK
  • 404 - object: Order not found
PUT /orders/{id} Update an order by ID

Parameters:

  • id (path, int) - The ID of the order

Responses:

  • 200 - object: Order successfully updated
  • 404 - object: Order not found
DELETE /orders/{id} Delete an order by ID

Parameters:

  • id (path, int) - The ID of the order

Responses:

  • 204 - string: Order successfully deleted
  • 404 - object: Order not found
