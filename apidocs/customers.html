API Documentation

GET /customers Get a list of customers

Parameters:

  • limit (query, int) - Limit the number of results
  • offset (query, int) - Offset the results

Responses:

  • 200 - array: OK
  • 403 - object: Unauthorized
GET /customers/{id} Get a customer by ID

Parameters:

  • id (path, int) - The ID of the customer

Responses:

  • 200 - object: OK
  • 404 - object: Customer not found
POST /customers Create a new customer

Responses:

  • 201 - object: Customer successfully created
  • 400 - object: Invalid request
