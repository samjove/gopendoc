API Documentation

GET /products/{id} Get a product by ID

Parameters:

  • id (path, int) - The ID of the product

Responses:

  • 200 - object: OK
  • 404 - object: Product not found
POST /products Create a new product

Responses:

  • 201 - object: Product successfully created
  • 400 - object: Invalid request
PUT /products/{id} Update a product by ID

Parameters:

  • id (path, int) - The ID of the product

Responses:

  • 200 - object: Product successfully updated
  • 404 - object: Product not found
DELETE /products/{id} Delete a product by ID

Parameters:

  • id (path, int) - The ID of the product

Responses:

  • 204 - string: Product successfully deleted
  • 404 - object: Product not found
