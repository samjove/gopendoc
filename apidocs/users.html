@route GET

Get a user by ID

Parameters

  • id (path, int) - The ID of the user

Responses

  • 200: object - OK
  • 404: object - User not found

@route POST

Post a new user

Responses

  • 201: object - User successfully created
  • 403: object - Bad Request
